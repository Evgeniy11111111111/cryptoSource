Средства разработки

Учитывая, что развитие QP ОС вышло на уровень, когда формируется пул прикладного ПО, приложений, была реализована задача разработки собственных средств разработки ПО для QP ОС:

  • Рабочая станция
  • Распределенная обработка информации
  • Управление сетевым взаимодействием
  • Средства разработки
  • Хранение и обработка данных

Поддерживаемые языки:

  • С
  • C++
  • C#
  • Python
  • JavaScript
  • ASP.NET + Nugets
  • PHP

Поддерживаемые библиотеки:

  • Qt
  • Avalonia
  • Другие библиотеки Open Source

Ведется портирование и разработка LLVM + clang, а также среды разработки.

Среда разработки

Среда разработки позволяет создавать приложения на языках «C» и «C#».

Комплект отладчиков

Отладчик управляемого кода предназначен для отладки управляемых .NET приложений и служб под управлением исполняющей машины Mono версии 3.1.x и выше. Отладка производится, как локально, так и удаленно (через сокеты).

Системный отладчик Используется для поиска ошибок в программных компонентах. Отладчик позволяет пошагово отслеживать работу программы, выявлять изменения системных переменных, состояния процессора во время работы программы.

Интерпретатор Python

Операционная система QP OС поддерживает интерпретатор языка Python версии 3.6.9. Python поддерживает такие модули, как socket, io, ssl. Также интерпретатор языка Python открывает новые возможности в разработке кроссплатформенного программного обеспечения.

Данный интерпретатор языка позволяет проводить:

  • Тестирование различных компонентов операционной системы
  • Тестирование сетевого взаимодействия
  • Реализацию различных модулей
  • Написание системных утилит, скриптов
