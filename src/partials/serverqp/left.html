Серверные технологии операционной системы QP ОС

Полнофункциональный набор серверных компонентов в составе QP ОС позволяет использовать ее как базовую операционную систему для построения защищенных информационных комплексов и систем обеспечения функционирования различных объектов.

Контроллер домена

Для обеспечения возможности формирования доменной сети рабочими станциями под управлением QP ОС в состав операционной системы входит подсистема управления системой каталогов, совместимая с Active Directory.

Подсистема позволяет настроить компьютер как в качестве контроллера домена, так и в качестве клиента в составе этого домена.

Почтовый сервер

Почтовый сервер QP Mail, функционирующий под управлением QP ОС, разработан с учетом требований по обеспечению защиты информации от несанкционированного доступа при обмене электронными сообщениями в системах, обрабатывающих конфиденциальную информацию. QP Mail может функционировать в комплексе как со средствами защиты линейки QP, так и с различными программно-аппаратными решениями по защите информации производства ООО НТП «Криптософт». В QP Mail предусмотрено два режима работы.

Режим сервера электронной почты

Сервер выступает в роли почтового сервера, обеспечивающего обмен электронными сообщениями между пользователями посредством протоколов электронной почты SMTP, POP3, IMAP4. Сервер позволяет обеспечить обмен электронными сообщениями между пользователями как в рамках одной обособленной ЛВС, так и с пользователями внешних систем электронной почты.

Режим шлюза электронной почты.

Сервер выступает в роли шлюза электронной почты, обеспечивающего обмен электронными сообщениями (через протокол SMTP) между серверами электронной почты двух и более обособленных ЛВС. Основной задачей шлюза является обеспечение контроля параметров передаваемых электронных сообщений между почтовыми системами.

Web-серверы

Web-сервер Megatron

В составе QP ОС реализован собственный веб-сервер, который представляет собой отдельную службу операционной системы и работает как сетевое приложение, производящее многопоточную обработку запросов по протоколам HTTP и HTTPS. Веб-сервер обеспечивает хостинг статического контента и поддерживает обработку запросов HEAD, POST, GET. При настройке сервера имеется возможность указать корневой каталог сервера, в котором непосредственно располагаются файлы статического контента сайта. Веб-сервер поддерживает интерфейс прикладного программирования CGI.

Web-сервер Nginx 1.14

В QP ОС портирован web-сервер nginx версии 1.14, позволяющий осуществлять взаимодействие с клиентами по протоколу HTTP 1.1 и совместимый с большинством современных браузеров.

Web-сервер Nginx поддерживает аутентификацию по протоколам NTLM, Kerberos и имеет положительное заключение ФСБ России на его применение совместно с операционной системой QP ОС для обработки информации с максимальной степенью секретности «совершенно секретно» включительно.

Web-сервер может быть использован в составе сложных систем обработки данных, реализованных на базе технологий CGI и FastCGI. Благодаря поддержке FastCGI и наличию в QP ОС платформ .NET и PHP, имеется возможность исполнения Web-приложений, разработанных для указанных платформ.

Интерпретатор PHP

В состав операционной системы QP OC входит интерпретатор PHP 7.3.20 с встроенными дополнительными средствами защиты. При построении защищенных веб-порталов PHP выступает в роли FastCGI-сервера и взаимодействует с веб-сервером Nginx на платформе операционной системы QP OC и веб-сервером IIS на платформе ОС Windows.

PHP 7.3.20 имеет положительное заключение ФСБ России на его применение совместно с операционной системой QP ОС для обработки информации с максимальной степенью секретности «совершенно секретно» включительно.

Функционально пакет PHP выполняет три задачи:

  • Анализ входного кода, компиляция и выполнение;
  • Предоставление функциональной части языка;
  • Взаимодействие с веб-сервером.

Механизмы безопасности PHP:

  • Сквозная аутентификация клиентов;
  • Контроль над запуском сценариев PHP;
  • Регистрация событий безопасности в системный журнал;
  • Расширение API PHP в соответствии со спецификацией PHP с целью интеграции с базовыми СЗИ.

Файловый сервер

В составе операционной системы QP ОС реализованы сетевые протоколы SMB версий 1 и 2. Они позволяют осуществлять подключение к разделяемым каталогам в составе локальной вычислительной сети.

FTP-сервер

В операционной системе QP ОС реализованы сервер и клиент службы протокола FTP, предназначенного для передачи файлов по TCP/IP сетям. Наличие данного функционала позволяет организовать хранение файлов на сервере и передачу файлов между сервером и клиентами.

DNS-сервер

В составе защищённой операционной системы QP ОС реализован собственный сервер системы доменных имён DNS. Приложение диспетчер DNS-сервера позволяет создать и настроить различные зоны (в том числе дочерние), серверы доменных имён (прямого и обратного просмотра, пересылки и др.). Созданные зоны поддерживают все основные типы ресурсных записей, включая записи адреса узла (A и AAAA), расположение службы (SRV), указателя (PTR), псевдонима (CNAME), авторитетного сервера имён (NS) и т.д.

Сервер видеоконференцсвязи

В составе операционной системы QP ОС разработаны все необходимые компоненты для управления видеоконференциями.

Функциональные возможности сервера:

  • Организация видеоконференций;
  • Запись сессий видеоконференций;
  • Планирование видеоконференций;
  • Управление пользователями и профилями видеоконференцсвязи.

Реляционные базы данных

Под управлением QP ОС функционируют две реляционные СУБД: разработанная НТП «Криптософт» СУБД QP DB и портированная СУБД с открытыми кодами PostgreSQL версии 9.6.2. Указанные СУБД частично совместимы между собой по формату запросов.

Система управления базами данных QP DB

СУБД QP DB интегрирована в комплексный механизм информационной безопасности QP ОС для надежного и безопасного хранения данных. СУБД QP DB полностью совместима по клиент-серверному протоколу с СУБД PostgreSQL, а в последних версиях эмулирует структуру системных таблиц и типов данных СУБД PostgreSQL. Данный подход позволяет осуществить переход на её использование большинства существующих программных продуктов, работающих с PostgreSQL с минимальными изменениями в программном коде.

В состав QP DB входит приложение для визуальной работы с СУБД, поддерживающее взаимодействие как с СУБД QP DB, так и с PostgreSQL.

Система управления базами данных PostgreSQL 9.6.2

PostgreSQL поддерживает большую часть стандарта SQL и предлагает множество современных функций:

  • Сложные запросы;
  • Внешние ключи;
  • Триггеры;
  • Изменяемые представления;
  • Транзакционная целостность;
  • Многоверсионность.

Особенности:

  • Сквозная аутентификация субъектов доступа;
  • Контроль целостности содержимого и дескрипторов безопасности объектов БД;
  • Обеспечение замкнутой программной среды;
  • Мандатный контроль доступа к строкам таблиц;
  • Аудит безопасности;
  • Блокировка субъектов доступа;
  • Средства администрирования с графическим интерфейсом.
